DayZ Server Manager Pro v3.0.0
DayZ Server Manager Pro — это профессиональное программное обеспечение для комплексного управления, контроля и администрирования DayZ серверов. Разработанное с учетом потребностей администраторов, приложение предоставляет удобный веб-интерфейс для мониторинга и настройки игровых серверов.

О разработчике
Данное программное обеспечение разработано и поддерживается для сообщества DayZ. При использовании кода или его модификации, пожалуйста, указывайте оригинального разработчика.
